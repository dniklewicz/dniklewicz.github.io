Frequently Asked Questions

In case of any problems please check if you have the newest version of UPS Power Monitor.


UPS Power Monitor application supports UPS devices connected to Synology NAS with USB cable.

Few steps are required in order to enable UPS network server on Sonylogy DSM:

  1. Open Control Panel

  2. Go to Hardware & Power

  3. Select UPS tab

  4. Select the checkmark next to Enable network UPS server

  5. Click on Permitted Synology NAS devices

  6. Enter your iPhone's IP address. Your device should have a static IP address or a DHCP reservation to work correctly in the future or you will need to change this setting in DSM every time your iOS device will have a new IP address given.

  7. Click OK
