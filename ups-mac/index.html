UPS Power Monitor application supports UPS devices connected to Synology NAS with USB cable.

Few steps are required in order to enable UPS network server on Sonylogy DSM:

  1. Open Control Panel

  2. Go to Hardware & Power

  3. Select UPS tab

  4. Select the checkmark next to Enable network UPS server

  5. Click on Permitted Synology NAS devices

  6. Enter your Mac's IP address. Your device should have a static IP address or a DHCP reservation to work correctly in the future or you will need to change this setting in DSM every time your Mac will have a new IP address given.

  7. Click OK

  8. In the UPS Power Monitor app, please provide IP address of the Synology NAS, port for the NUT service (3493 is a default one) and click Save.

  9. If required, please use "ups" as the UPS Server Username and click the Save button again.

In order to connect to a NUT/UPS Server running on the QNAP NAS please follow the steps below:

  1. Open: Control Panel -> System -> External Device -> UPS.

  2. Click "Add" and enter your Mac's IP address. Your device should have a static IP address or a DHCP reservation to work correctly in the future or you will need to change this setting in QTS every time your Mac will have a new IP address given.

  3. Enter your QNAP's IP Address and leave the default port number. Click the Save button.

  4. If required, please use "qnapups" as the UPS Server Username and click the Save button again.

  5. Click Save.
