Frequently Asked Questions

In case of any problems please check if you have the newest version of the application.


I have an active Intellimote Pro subscription but additional features are locked.

Please go to Settings -> Intellimote Pro and tap Restore button located in the upper right corner of the screen. After that, Intellimote will restore all active subscription data from the App Store.


Which smart speakers are fully compatible with Intellimote?

All devices supporting HEOS by Denon and Yamaha MusicCast technologies are supported. Please note that HEOS Enabled Denon/Marantz AVRs are not fully compatible with HEOS technology and are supported only partially..


Which TV devices are fully compatible with Intellimote?

Please look at this list.


I cannot connect to my TV

Open your TV's settings (usually you need to press the Home Button on your TV's remote and then select Settings) and make sure that following is set:

• Make sure your TV is connected to your local network
• Settings → Network → Home Network Setup → Remote Start => On
• Settings → Network → Home Network Setup → Remote Device / Renderer -> Remote Device / Renderer => On
• Settings → Network → Home Network Setup → IP Control → Authentication => Normal or Pre-Shared Key or Normal and Pre-Shared Key
• Settings → Network → Home Network Setup → IP Control → Pre-Shared Key => 1111 or any other key
• Settings → Network → Home Network Setup → IP Control → Simple IP Control => On • As a last step please restart your router


Should I allow incoming connections when asked by firewall?

Yes, Remote for BRAVIA requires incoming connections to automatically discover compatible devices and to receive updates when their state changes (like volume, mute, content).

Yamaha® MusicCast®:


• All MusicCast® smart speakers
• MusicCast® Enabled AVR devices
